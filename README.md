# Product — Agile Product Management Compendium

Recherche- und Visualisierungs-Repo für moderne agile Produktentwicklung im
Enterprise-Kontext (500+ Personen, Multi-BU, Compliance, Legacy-Integration).

## Zielsetzung

1. **Vergleich** moderner und klassischer agiler Methoden — bewusst *ohne* das
   Skalierungs-Schwergewicht (SAFe/LeSS), mit Fokus auf den Product-Operating-Model-Stack
   (Cagan + Team Topologies + Continuous Discovery).
2. **Visualisierung** eines End-to-End-Outcome-Loops als Referenz-Zyklus für
   komplexe Produktorganisationen.

## Struktur

```
docs/
  specs/         Projekt-Spec und Design-Entscheidungen
  cycle/         Haupt-Visualisierung: Enterprise Outcome-Loop
  methods/       Methoden-Profile
    classical/   Scrum, Kanban, XP, Crystal
    discovery/   Shape Up, Dual-Track, Continuous Discovery, Lean Startup, JTBD
    modern/      Team Topologies, FAST, Outcome Roadmapping, POM, AI-augmented
  comparison/    Vergleichsmatrix
```
